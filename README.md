# Blockfinder

## Setup virtualenv
`virtualenv myenv -p python3`

## Clone the repo
`git clone https://github.com/brodia10/Blockfinder.git`

## Install requirements
`pip3 install -U -r requirements.txt`

## Start Server
`python manage.py runserver`

or

`make build`
